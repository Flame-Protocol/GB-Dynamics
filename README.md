# GB Dynamics Inc. - Verification Portal

## Quick Deploy

```bash
cd ~/Downloads
unzip GB_Dynamics_Verification.zip
cd gb-dynamics

git init
git remote add origin https://github.com/flame-protocol/GB-Dynamics.git
git add .
git commit -m "GB Dynamics Verification Portal"
git branch -M main
git push -f origin main
```

## Enable GitHub Pages

1. Go to: https://github.com/flame-protocol/GB-Dynamics/settings/pages
2. Source: **Deploy from a branch**
3. Branch: **main** / **(root)**
4. Save
